🔬 RMSNorm改进原理

LayerNorm (原始)
LN(x) = γ · (x - μ) / σ + β
需要计算均值μ和方差σ
RMSNorm (改进)
RMS(x) = γ · x / √(E[x²])
只需计算均方根，无需均值和偏置

🧬 SwiGLU门控机制

ReLU FFN (原始)
FFN(x) = max(0, xW₁ + b₁)W₂ + b₂
